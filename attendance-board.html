Click a cell to mark that day as present for that employee (click again to unmark). The cell's color is the running count of present days that week, in Monday→Sunday order: their 1st present day turns color 1, 2nd present day turns color 2, and so on up to 7 colors for a full week present. The number shown in each colored cell is that running count. This board's data is shared — anyone you send this artifact link to sees the same employees and attendance, refreshing automatically every few seconds to pick up others' changes.
